# Multiplix

A simple multiplication game for children, you can try it on [multiplix.labs.gkdv.net](multiplix.labs.gkdv.net)


## Install
```bash
$ make build
$ make install
```

## Run (electron version - desktop gui)
```bash
$ make gui
```


## Serve static html ui (nginx)
```bash
$ make serve-html
```

## Generate an android APK (cordova)
```bash
$ make android
$ make build
```
